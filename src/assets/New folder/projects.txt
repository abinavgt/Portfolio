Projects:

1. Spectrum Lens(Chrome Extension)
Spectrum Lens is a lightweight, high-performance Chrome extension designed for designers and developers to capture hex codes instantly from any webpage. Built using JavaScript and the EyeDropper API, it allows users to pick colors with a single click, automatically saving them to a persistent history via Local Storage. The extension features a clean, responsive interface with one-click clipboard copying, a clear-all history function, and a 3-column grid layout for organized color management
Tech Stacks: JavaScript, HTML, CSS, Loaded Packages
Github link: https://github.com/abinavgt/SpectrumLens

2. Advanced Computer Vision(ML/AI)
Advanced Computer Vision is an interactive repository featuring high-performance AI applications built with OpenCV and MediaPipe. It enables real-time gesture-controlled gaming (Snake, Fruit Ninja) and sophisticated human analysis modules, including face mesh tracking and pose estimation
Tech Stacks: Python, OpenCv, MediaPipe, PyGame
Github link: https://github.com/abinavgt/Advanced-Computer-Vision

3. ATM Simulator(Full-Stack)
ATM Simulator is a robust banking application designed to replicate real-world ATM operations with algorithmic precision. Built with Python and Flask, it features a sophisticated engine that calculates real-time multi-currency denomination breakdowns (INR, USD, EUR, etc.). The platform provides a secure environment for PIN-based authentication, balance inquiries, and instant mini-statement generation, delivering a seamless and responsive financial experience
Tech Stacks: Python, JavaScript, Flask, MySQL, HTML, CSS
Github link: https://github.com/abinavgt/ATM-Simulator

4. Heart Stroke Prediction (ML)

Tech Stacks: Python, TensorFlow, scikit-learn, joblib, Streamlit
Github link: https://github.com/abinavgt/Heart-Failure-Prediction
